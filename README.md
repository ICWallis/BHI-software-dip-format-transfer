Various borehole image (BHI) log software generates outputs and expects imports with different conventions and formats. This repo contains methods to improve interoperability between software packages.

Ideally, all BHI software should follow the SPWLA Dip Exchange Format (download from https://www.spwla.org/SPWLAArchived/SPWLA/Chapters_SIGs/SIGs/Borehole_Imaging/Borehole_Imaging.aspx). However, in my view the LAS 2.0 format would be more universal than the LAS 3.0 format they recommend.  

The initial focus of this project is on Geolog and WellCAD. Get in touch (irene@cubicearth.nz) if you use other software and would like to contribute to this project.
